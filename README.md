jqFastreeview
=============

### Informations ###
Version 0.1.
Ce plugin a pour vocations à mettre en place rapidement et simplement une arborescence à partir d'une suite de balise '<ul> et <li>'.
Crée à partir de jQuery 1.9.1.

## Dépendance ##
